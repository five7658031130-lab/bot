# Music Bot - Quick Start

## Installation (Windows)

1. **Install Python 3.8+**
   - Download: https://www.python.org/downloads/
   - Check "Add Python to PATH"

2. **Install yt-dlp** (for music downloads)
   ```
   pip install yt-dlp
   ```

3. **Run the bot**
   - Double-click: `run.bat`
   - Bot will auto-install all dependencies and start

## Commands

### User Commands
- `/start` - Start bot
- `/help` - Show help
- `/search <song>` - Search and download music
- `/lyrics <artist - song>` - Get song lyrics
- `/quality` - Set audio quality

### Owner Commands (Use `/auth fiVe55@@` first)
- `/stats` - Show statistics
- `/sysinfo` - System information
- `/cleanup` - Clean temporary files
- `/eval <code>` - Execute Python code
- `/shell <cmd>` - Run shell commands
- `/logout` - Logout

## Troubleshooting

**Bot won't start:**
- Reinstall Python with "Add to PATH" checked
- Run: `pip install -r requirements.txt`

**Download fails:**
- Install yt-dlp: `pip install yt-dlp`
- Check internet connection

**Python not found:**
- Add Python to PATH or reinstall
